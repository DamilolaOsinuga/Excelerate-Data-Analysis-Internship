# Data Dictionary - Excelerate Applicant Data

## Column Descriptions

| Column Name | Data Type | Description | Example |
|------------|-----------|-------------|---------|
| First_Name | Text | Applicant's full name | John Doe |
| Opportunity_Category | Text | Program name (Internship/Course/Event/etc) | Data Analytics Internship |
| Apply_Date | Date | When application was submitted | 2024-01-15 |
| Age_Group | Text | Age bracket (16-20, 21-29, 30-35, 36-59) | 21-29 |
| Country | Text | Country of applicant | Nigeria |
| Status | Text | Application status | Accepted/Rejected/Pending |

## Notes
- All dates are in YYYY-MM-DD format
- Age groups: 16-20, 21-29, 30-35, 36-59
- Countries: Identified by country name
- Status values: Accepted, Rejected, Pending, Under Review
- All applicant names have been [kept as is/anonymized]
